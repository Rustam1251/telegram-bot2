# ⚡ Оптимизации генерации уроков

## Проблема
Бот зависал на генерации уроков (долгий ответ от Groq API)

## Решения

### 1. Сокращение длины уроков
- **Было**: 1200-1500 знаков
- **Стало**: 800-1000 знаков
- **Результат**: Быстрее генерация, меньше токенов

### 2. Уменьшение max_tokens
- **Уроки**: 2000 → 1500 токенов
- **Вопросы**: 500 → 400 токенов
- **Результат**: Быстрее ответ API

### 3. Добавлены таймауты
- **Генерация урока**: 45 секунд
- **Ответ на вопрос**: 30 секунд
- **API запрос урока**: 30 секунд
- **API запрос вопроса**: 20 секунд
- **Результат**: Бот не зависает навсегда

### 4. Оптимизация retry
- **Было**: 3 попытки с ожиданием 2-10 сек
- **Стало**: 2 попытки с ожиданием 1-5 сек
- **Результат**: Быстрее обработка ошибок

### 5. Улучшенные сообщения
- Добавлен индикатор времени: "Это займёт 10-15 секунд..."
- Понятные сообщения об ошибках таймаута
- Предложения попробовать позже

## Результаты

✅ Генерация урока: ~10-15 секунд (было ~30-60 сек)
✅ Ответ на вопрос: ~5-10 секунд (было ~20-30 сек)
✅ Нет зависаний при перегрузке API
✅ Понятная обратная связь пользователю

## Дополнительные улучшения

### Логирование
- Логируются таймауты
- Логируются все ошибки генерации
- Можно отслеживать проблемные темы

### Обработка ошибок
- Отдельная обработка TimeoutError
- Отдельная обработка API ошибок
- Graceful degradation

## Рекомендации

1. **Мониторинг**: Следите за логами на наличие частых таймаутов
2. **Groq API**: Проверяйте лимиты и квоты
3. **Темы**: Если какая-то тема часто вызывает таймауты - упростите промпт
4. **Пользователи**: При высокой нагрузке можно увеличить rate limit период

4. Убедиться, что сообщения понятны пользователю
