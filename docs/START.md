# 🚀 Быстрый запуск EGOIST ACADEMY

## Проблема с подключением

Бот не может подключиться к api.telegram.org из-за блокировки в вашем регионе.

## Решения:

### 1. Использование VPN (Рекомендуется)
- Включите VPN на компьютере
- Запустите: `python main.py`

### 2. Использование прокси
Добавьте в `.env`:
```
PROXY_URL=socks5://user:pass@host:port
```

Обновите `main.py` для использования прокси.

### 3. Деплой на VPS сервер
Используйте сервер за пределами РФ:
```bash
# На сервере
git clone your_repo
cd egoist_academy
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
# Отредактируйте .env
python main.py
```

## Проверка работы

## Тестирование

1. Откройте Telegram
2. Найдите вашего бота
3. Отправьте `/start`
4. Пройдите регистрацию
