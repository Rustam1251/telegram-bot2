# EGOIST ACADEMY Bot

Telegram-бот для обучения в традициях old money. Полностью автономная система с 12 модулями, генерацией уроков через AI и интерактивными тестами.

## 🎯 Возможности

- **12 образовательных модулей**: Этикет, Стиль, Искусство, Финансы, Путешествия, Право, Вино, Психология, Ораторство, Лидерство, Время и досуг, Политика
- **AI-генерация уроков**: Каждый урок генерируется персонально через Groq API
- **Интерактивные тесты**: Проверка знаний после каждого урока
- **Система прогресса**: Отслеживание уровня и достижений
- **Избранное**: Сохранение интересных уроков
- **Персональные настройки**: Изменение уровня и интересов
- **Следующий урок**: Генерация нового урока из той же подтемы

## 📋 Требования

- Python 3.10+
- Telegram Bot Token
- Groq API Key
- SQLite3

## 🚀 Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/yourusername/egoist-academy.git
cd egoist-academy
```

### 2. Создание виртуального окружения

```bash
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Конфигурация

Создайте файл `.env`:

```env
BOT_TOKEN=your_telegram_bot_token
GROQ_API_KEY=your_groq_api_key
GROQ_BASE_URL=https://api.groq.com/openai/v1
GROQ_MODEL=mixtral-8x7b-32768
DATABASE_PATH=academy.db
LOG_FILE=bot.log
```

### 5. Запуск

```bash
python3 main.py
```

## 🔧 Использование

### Для пользователя

1. Напишите боту `/start`
2. Пройдите регистрацию (имя, интересы, уровень)
3. Выберите модуль из меню
4. Выберите тему для изучения
5. Получите персональный урок
6. Пройдите тест для проверки знаний
7. Оцените урок (👍/👎)

### Для разработчика

#### Добавление нового модуля

1. **quotes.py**: Добавьте цитату и описание
```python
"new_module": "«Цитата» — Автор.",
"new_module": "Описание модуля..."
```

2. **keyboards.py**: Добавьте кнопку и подтемы
```python
("🆕 Новый модуль", "module_new_module"),
"new_module": [
    ("Тема 1", "topic_new_module_1"),
    ("Тема 2", "topic_new_module_2"),
]
```

3. **utils.py**: Добавьте в словари
```python
"topic_new_module_1": "Тема 1",
"topic_new_module_1": "new_module",
```

## 🐛 Решение проблем

### Бот не отвечает
- Проверьте BOT_TOKEN в .env
- Убедитесь в подключении к интернету
- Проверьте логи в bot.log

### Ошибки генерации уроков
- Проверьте GROQ_API_KEY
- Убедитесь в лимитах API
- Проверьте логи

### Проблемы с БД
- Удалите academy.db и перезагрузите бота
- Проверьте права доступа к файлам
