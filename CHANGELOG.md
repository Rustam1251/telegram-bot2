## Структура базы данных

### Таблицы:
1. `users` - пользователи
2. `lessons` - уроки
3. `progress` - прогресс обучения
4. `favorites` - избранные уроки
5. `user_actions` - логи действий
6. `knowledge_base` - база знаний

## Основные функции

### Пользователи
- `add_user()` - регистрация
- `get_user()` - получение данных
- `update_user_level()` - обновление уровня

### Уроки
- `save_lesson()` - сохранение урока
- `get_lesson()` - получение урока
- `get_user_lessons()` - история уроков

### Прогресс
- `add_progress()` - добавление записи
- `update_progress_score()` - обновление результата
- `get_user_progress()` - статистика
- `check_and_update_level()` - автоповышение уровня

### Избранное
- `add_favorite()` - добавить в избранное
- `remove_favorite()` - удалить из избранного
- `get_favorites()` - список избранного

### Рекомендации
- `get_smart_recommendation()` - умные рекомендации на основе прогресса

### Логирование
- `log_user_action()` - логирование действий

## Известные проблемы

1. **Блокировка Telegram API** - требуется VPN или VPS
2. **Windows** - могут быть проблемы с кодировкой (используйте UTF-8)
